Need help re-entering InfraBox?

What kind of project do you want to start with?

Start with a GitHub Project
Step 1: Connect your repository
Go to your GitHub repository overview page:
Select the repositories you want to connect with InfraBox:

Step 2: Book a machine configuration

You can choose from the following configurations:
Name CPU RAM Storage
vm-mini-1024 0.5 Core 1024 MB 10 GB
vm-1-2048 1 Core 2048 MB 20 GB
vm-2-4096 2 Cores 4096 MB 30 GB
vm-4-8192 4 Cores 8192 MB 40 GB
For pricing and subsriptions have a look at this page.

Step 3: Set up your configuration

The heart of your InfraBox project is the infrabox.json file. In this document you define your jobs, their dependencies and resources.
This article explains how to set up the infrabox.json file.
Assign one of the subscribed machine configurations to each of your jobs.

Step 4: Run your project on InfraBox

Put your infrabox.json file into the root directory of your GitHub project.
Now every time you push to you repository InfraBox will run your jobs and visualizes their results.
